This project was created at Plovdiv University during lectures.

- Adapter Pattern
- Chain Of Responsibility Pattern
- Command Pattern
- Composite Pattern
- Decorator Pattern
- Facade Pattern
- Factory Method Pattern
- Iterator Pattern
- Observer Pattern
- Proxy Pattern
- Singleton Pattern
- State Pattern
- Strategy Pattern
